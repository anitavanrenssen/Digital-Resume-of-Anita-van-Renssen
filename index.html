Netflix clone

A Netflix clone app built with the use of Vue's object oriented programming system. The app has a login page which displays the user's name across all pages. It also includes a 'coming soon' banner and available movies filtered by genre. Movies can be added to a watch list and removed using localStorage and JSON. The watch list page includes a searchbar which filters the movies instantly according to the user input.

  • HTML
  • CSS
  • JavaScript
  • Vue.js

A single-page application that helps users keep track of various tasks. JavaScript and localStorage was used to create this CRUD app where tasks and an optional date can be created, read, updated and deleted. The tasks can also be sorted alphabetically.
